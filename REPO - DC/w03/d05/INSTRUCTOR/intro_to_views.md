# Intro to Views

**Learning Objectives**

- What is a view, where do they belong?
	- "views" folder
	- ".erb" extension
- What are static files, where do they belong?
	- "public" folder
	- these are unrendered files
- What is ERB?
	- `layout.erb` / yeild
	- control flow
	- output
- What is *localization*, how do we accomodate it?
	- All display text remains *in* the view.
